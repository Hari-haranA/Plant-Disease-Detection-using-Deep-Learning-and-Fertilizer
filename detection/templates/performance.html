ResNet Model Architecture - Summary

ResNet (Residual Network) is a deep learning model designed to overcome vanishing gradient issues, allowing training of very deep neural networks. It introduces residual connections, enabling information to bypass layers, improving accuracy and convergence.

Key Features of ResNet

  • Residual Blocks: Each layer feeds into the next and also connects to layers 2-3 hops away.
  • Batch Normalization & ReLU: Enhance training speed and stability.
  • Max Pooling: Reduces spatial dimensions while preserving key features.
  • Deep Convolutional Layers: Extract hierarchical features from input images.

Why ResNet?

  • Prevents overfitting.
  • Avoids vanishing gradient problems.
  • Enables training of very deep networks.

The ResNet9 Architecture Consists of:

  • Convolutional Layers: Extracting features.
  • Residual Blocks: Enhancing gradient flow.
  • Fully Connected Layer: Classifying images into 38 categories.

This architecture ensures efficient and accurate image classification, making it ideal for plant disease detection. 🚀

Conclusion

ResNets perform significantly well for image classification when some of the parameters are tweaked and techniques like scheduling learning rate, gradient clipping and weight decay are applied. The model is able to predict every image in test set perfectly without any errors !!!!
